### [agency\_code](#agency_code)
**Description:** The integer code corresponding to an institution's regulatory agency
**Values:**
- 1: OCC
- 2: FRB
- 3: FDIC
- 5: NCUA
- 7: HUD
- 9: CFPB

### [id\_2017](#id_2017)

**Values:**
- Various values
